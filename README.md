# DraftHQ

A multiplayer fantasy draft and league-experience platform focused on live draft night.

## Current Features

- Local draft room MVP
- Snake draft board
- Manual picks
- Recent picks
- Undo
- Supabase schema foundation
- Sleeper import
- Realtime picks
- Team ownership
- Commissioner controls

## Current Focus

- Reliable multiplayer drafting
- Timer synchronization
- Refresh/reconnect recovery
- Mobile support
- Clearer on-the-clock presentation

## Long-Term Vision

DraftHQ gives fantasy leagues their own identity, history, branding, and draft-night presentation.

## Goal

League-ready multiplayer draft room by September 2.
